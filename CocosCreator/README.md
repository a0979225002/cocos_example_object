# CocosCreator Effect第三方控件
##当前功能：
* 支持拖拽方式添加Effect效果到Creator项目  
* 支持Creator WebGL 和 Native原生版本
* 新增加20个Shader特效
* 新增加160个Shadertoy的特效

##日志： 
* 优化Shader代码的写法，更易于阅读修改 

##ToDo:
* textureCube的支持
* 对声音的支持
* BuffABC模式的支持
